How to play:1. Open up terminal and the location to which the file is in2. type make in the terminal to compile the Tic Tac Toe game3. Then the type ./FinalProject to run the actual game4. From there you will be greeted with a screen on the terminal that is set up like atraditional Tic Tac Toe game but Letters going up vertically on the side and numbersgoing across horizontally on the top5. From here all you have to do is type in the coordinates of the spot you want to useEX: you would type A1 for the top left spot and C3 for the bottom right spot6. Rules are the exact same as the regular Tic Tac Toe game and whenever you win or loseor tie you will see a text saying who won or if there was a tie.------------------------------------------------------------------------------------------
